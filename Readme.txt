== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `post-ajax-slider` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php require_once("wp-content/plugins/post-ajax-slider/post-viewer.php"); ?>` in your templates
